# Sound Effects

Add these four sound files to this directory:

1. **hadouken-charge.mp3** ✅ - Charging/powering up sound (loops)
   - Should play continuously while charging
   - Energy building sound

2. **hadouken-blast.mp3** ✅ - Throwing/blast sound
   - Plays once when you release the hadouken
   - Classic "HADOUKEN!" sound

3. **hadouken-impact.mp3** ✅ - Hit/impact sound
   - Plays when the blast hits the text
   - "Faaa" or impact sound effect

4. **street-fighter-bgm.mp3** - Background music (loops)
   - Plays continuously in the background at 30% volume
   - Street Fighter stage theme or similar fighting game music

## Where to get sounds:

**Option 1: Street Fighter sound rips**
- Search for "Street Fighter hadouken sound effects"
- Download from game sound archives or YouTube

**Option 2: Freesound.org (royalty-free)**
- Charging: Search "energy charge" or "power up"
- Blast: Search "whoosh" or "energy blast"
- Impact: Search "hit" or "impact punch"

**Option 3: Generate with AI**
- Use tools like ElevenLabs or similar to generate sound effects
- Describe: "energy charging sound", "hadouken blast", "hit impact"

## File format:
- MP3 format preferred
- Keep file sizes reasonable (< 500KB each)
- Sample rate: 44.1kHz recommended
